Defines a structure to hold the necessary information to execute class files.

The main struct of this module is a JavaVirtualMachine. It holds all data necessary for the execution of class file.

To setup the virtual machine, a call to initJVM() is made. When execution is finished, a call to deinitJVM() will release any resources associated with the JVM.

The class holding a "public static void main(String[])" method must be resolved before being executed, which can be done with resolveClass().

Any other class files that the entry class file uses is resolved automatically during run-time.
